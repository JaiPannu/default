# Why we made it
There are too many things that can go wrong when completing a robotics challenge. Judges need to keep track of every robot with maximum fairness which introduces too much human factor. We wanted to keep an eye out for things that go wrong and especially those that go right so we made the Real1

# What it is
A hardware to blockchain bridge that creates a trustless system of keeping score authentication so that the telemetry data goes straight to a universal ledger for judges to look at
